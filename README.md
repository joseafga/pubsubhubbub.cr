# PubSubHubbub.cr

PubSubHubbub subscriber library written in Crystal.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     pubsubhubbub:
       github: joseafga/pubsubhubbub.cr
   ```

2. Run `shards install`
